About

I am a senior software engineer working at InfoMedix Pty Ltd since 2019. Before that, I was a software researcher partaking numerous national and EU R&D projects such as WWTF CONTAINER, EU FP7 INDENICA, EU FP7 COMPAS, and SemBiz. The main focus of my research and development are including software architecture, distributed systems, SOAs, event-driven architectures, model-driven engineering, and application of formal methods.

  • Service-Oriented Architectures and Process-driven SOAs / Process Management (BPMN, BPEL)
  • Model-driven Software Development
  • Domain-Specific Languages (DSLs)
  • Event-Driven Systems
  • JavaScript, React.js, Material UI, HTML, (S)CSS, Node.js
  • Web and RESTful services, WSDL, JAX-WS, JAXB, JAX-RS / JSON (Jackson)
  • Java | Jakarta EE (WildFly/JBoss), Spring Framework
  • AWS: VPC, Container (EC2, EKS), S3, Aurora, API Gateway, Serverless / Microservices (Lambda, Step Functions)
  • Database servers, ORM (JPA / Hibernate), SQL
  • Apache CXF, JMS / Apache ActiveMQ
  • Java, Scala, Groovy, Python, C, Shell Scripting
  • Unit Testing (JUnit, Mockito, Jest), Selenium
